# Arch gen: rounded-corner cantilever balcony tower (bpy/bmesh)

Generates an Art-Deco/streamline residential tower whose corner is a rounded drum with continuous cantilevered balcony bands wrapping each floor, fronted by curved railings. Parameters: floor count, corner radius, balcony slab overhang, railing height/style, floor height, and window rhythm along the curve. Inspired by pinterest_657103401931588863.jpg.
