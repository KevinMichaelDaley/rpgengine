# Arch gen: tall multi-pane steel factory window (bpy/bmesh)

## Description

Generates a tall industrial steel-frame window wall: a slim mullion/muntin grid of many small lites, an outer frame, optional projecting hopper/awning vent sashes, and a masonry sill and header. Parameters: overall width/height, lite rows/columns, mullion thickness, operable-sash rows, glass inset depth, and an optional segmental-arch or keystoned head. Inspired by pinterest_657103401931588710.jpg, pinterest_657103401931588733.jpg, and pinterest_657103401931588823.jpg.
