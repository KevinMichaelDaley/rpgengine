# §5.2.1 Per-bone collision body inspector

Inspector panel for selecting and configuring collision shapes per bone in the prefab editor.

## Deliverables

- Shape type dropdown per bone: none / box / sphere / capsule / convex hull
- Per-shape parameter fields:
  - Box: half-extents (x, y, z)
  - Sphere: radius
  - Capsule: radius + height
- Auto-fit button: compute tight-fitting shape from mesh vertices weighted to the bone (weight > 0.1)
- Mass field (kg) with optional density mode (compute mass from shape volume)
- Collision group and mask flag checkboxes
- Shape wireframe rendered in collision overlay, updates live as parameters change
- Store shape descriptor in bone_collider_desc_t (already exists in fskel v2 COLL chunk)
- Undo/redo for shape changes

## Key files

- src/editor/scene/scene_ui_bone_inspector.c — add collision body section
- include/ferrum/animation/constraint_params.h — bone_collider_desc_t
- src/editor/scene/scene_viewport_collision_overlay.c — render shape wireframes
