# Arch gen: steel zigzag fire escape with balconies and drop ladder (bpy/bmesh)


## Description

Generates a wall-mounted steel zigzag fire escape: stacked landing platforms with expanded-metal or bar-grate floors, angle-iron railings, diagonal stair flights alternating side to side between floors, stringers, and a counterweighted drop ladder at the bottom. Parameters: floor count, landing depth/width, stair rise/run and slope, baluster spacing, grate pattern, standoff distance from wall, rust/weathering amount, drop-ladder deployed vs stowed. Inspired by pinterest_657103401931588785.jpg and pinterest_657103401931588869.jpg.
