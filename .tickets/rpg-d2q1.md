# Arch gen: setback high-rise office tower (bpy/bmesh)

Generates a downtown office/commercial tower built from stacked prismatic volumes with vertical setbacks, a regular curtain-wall/window grid per floor, and a flat or stepped crown. Parameters: floor count, floor height, footprint width/depth, number and height of setbacks, window module size, mullion depth, and crown style.
