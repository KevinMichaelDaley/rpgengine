# Arch gen: Brick tenement facade with stone-lintel double-hung windows (bpy/bmesh)


Generates a load-bearing brick tenement facade: coursed brick wall with a regular grid of window openings, projecting stone lintels and sills, and optional water-table/belt courses. Key params: floor count, window grid columns/spacing, opening width/height, lintel and sill projection, brick course height and bond jitter, mortar-recess depth, weathering/soot gradient. Inspired by pinterest_657103401931588768.jpg and pinterest_657103401931588859.jpg.
