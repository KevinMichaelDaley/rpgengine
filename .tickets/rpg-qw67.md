# Arch gen: Wall-mounted AC condenser unit cluster (bpy/bmesh)


Generates a cluster of wall-mounted split-system AC condenser units bolted to a facade on brackets, at randomized heights with dangling refrigerant lines and drip stains. Key params: unit box dimensions and count, cluster bounding region, per-unit height/offset jitter, bracket depth, louver-fin subdivisions, line-set droop, rust/drip-stain intensity.
