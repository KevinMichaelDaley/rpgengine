# Arch gen: overhead tangled utility cable web (bpy/bmesh)


Generates the tangled overhead utility-cable web spanning between buildings and poles: catenary-sagging bundles of wires, junction/transformer boxes, splice knots, and drop lines, generated along and across an alley volume. Parameters: anchor points, bundle count, sag amount, sag jitter, junction-box density, and droop-line probability. A set-dressing overlay for alleys. Inspired by pinterest_657103401931588844.jpg, pinterest_657103401931588800.jpg and pinterest_657103401931589018.jpg.
