# Arch gen: street vending-machine bank (bpy/bmesh)


Generates a row/bank of illuminated vending machines: box bodies with a glazed product window, a lit top header sign, coin/selection panel, and a recessed dispensing tray, lined up with slight depth and height jitter. Parameters: machine count, individual widths, glow color per unit, header-sign toggle, and a grime/sticker decay flag. Inspired by pinterest_657103401931589018.jpg.
