# Arch gen: exterior straight-run stucco stair with pipe rail (bpy/bmesh)


Generates an exterior straight-run access stair built as a solid stucco-clad stringer with cast treads and a simple round-pipe handrail on steel posts. Parameters: total rise, run length, tread count, stair width, landing size, rail height and post spacing, and a soffit-thickness control. Attaches to an apartment facade at an arbitrary door height. Inspired by pinterest_657103401931572033.jpg and pinterest_657103401931571992.jpg.
