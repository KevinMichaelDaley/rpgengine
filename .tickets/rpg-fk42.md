# Arch gen: coffered concrete beam ceiling (bpy/bmesh)

## Description

Builds a heavy cast-concrete coffered ceiling: a deep primary/secondary beam grid forming rectangular sunken coffers, with chamfered beam soffits and optional form-board plank texture on the faces. Parameters: bay counts in X/Y, beam depth and width, coffer inset depth, chamfer size, and board-mark toggle for weathered brutalist surfaces. Inspired by pinterest_657103401931588559.jpg (bunker/tunnel interior).
