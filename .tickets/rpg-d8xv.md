# Arch gen: strung tarp alley canopy (bpy/bmesh)

Generates a strung tarp/cloth alley canopy: a sagging fabric sheet (or several) stretched across a narrow alley between opposing buildings, tied to hooks/wires with catenary sag and pooling folds, filtering light below. Key params: span width/length, sag depth, tie-point count and spacing, number of overlapping sheets, wrinkle/fold noise seed, edge fray. Inspired by pinterest_657103401931588744.jpg (Varanasi tarp-covered lane).
