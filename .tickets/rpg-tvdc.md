# Arch gen: Steel-truss barrel-vault glazed atrium skylight (bpy/bmesh)


Generates a steel-truss barrel-vault glazed atrium skylight: a semicircular vault spanning a hall, built from arched primary ribs, longitudinal purlins, diagonal web bracing, and glazing panels (with optional stained-glass tympanum ends). Key params: span/radius, bay length and rib count, purlin subdivisions, web-truss depth/pattern, glazing-panel size, tympanum infill toggle, spring-line height. Inspired by pinterest_657103401931588901.jpg (dead-mall atrium).
