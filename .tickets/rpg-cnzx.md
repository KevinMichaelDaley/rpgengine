# Arch gen: dense tenement lightwell facade (bpy/bmesh)

## Description

Generates the tall grimy facades that line a narrow interior airshaft/lightwell between packed tenements: stacked irregular window openings, projecting drip ledges, protruding pipes and conduit, drooping wire spans crossing the gap, and small lean-to roofs at the base. Parameters: facade height/floors, well width, window irregularity, pipe/wire density, staining intensity, and opposing-wall offset. Inspired by pinterest_657103401931588823.jpg (Kowloon-style alley).
