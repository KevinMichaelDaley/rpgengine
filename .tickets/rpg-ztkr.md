# Arch gen: projecting theater marquee with letterboard (bpy/bmesh)

Generates a cantilevered box marquee projecting over a corner entrance, with an angled/chamfered front, changeable letterboard panels on each face, a soffit underside with light sockets, and a crowning cornice. Parameters: projection depth, marquee height, chamfer angle (corner vs flat), letterboard rows, soffit light count, and support tie-rod toggle. Inspired by pinterest_657103401931571921.jpg.
