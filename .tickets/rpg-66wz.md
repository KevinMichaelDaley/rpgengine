# Arch gen: cantilevered motel balcony walkway stack (bpy/bmesh)

## Description

Builds a mid-century motel/apartment block with stacked floor slabs that cantilever outward as continuous exterior balcony walkways, each edged by a thin slab fascia and a simple picket or pipe guardrail. Parameters: floor count, slab overhang depth, floor-to-floor height, cantilever taper per level, railing style/height, and column/post placement under the lowest deck. Inspired by pinterest_657103401931571905.png (Tropicana Motel).
