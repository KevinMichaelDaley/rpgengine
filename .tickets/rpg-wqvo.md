# Arch gen: cylindrical barber-pole sign column (bpy/bmesh)

Generates a wall-mounted or freestanding cylindrical sign column wrapped in helical stripe bands (classic barber pole), with rounded end caps and a bracket mount. Parameters: cylinder height/radius, stripe count and helix pitch, cap style, mount type (wall bracket vs post), and stripe colors. Inspired by pinterest_657103401931571902.jpg.
