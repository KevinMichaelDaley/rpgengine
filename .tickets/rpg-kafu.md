# Arch gen: Beaux-Arts multi-story commercial block with cornice (bpy/bmesh)


Generates a large early-20th-century Beaux-Arts / Renaissance-Revival commercial block: a repetitive multi-story window grid organized into base/shaft/crown tripartite composition, terminated by a deep bracketed cornice. Key params: bays wide and floors tall, storey height, window proportions per tier, spandrel treatment, cornice depth/bracket spacing, corner-pier width, ground-floor storefront band height. Inspired by pinterest_657103401931588582.jpg (One Wilshire block).
