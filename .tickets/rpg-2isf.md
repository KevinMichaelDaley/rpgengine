# Arch gen: exposed HVAC ductwork ceiling run (bpy/bmesh)

Generates an exposed basement/utility ceiling of round and rectangular HVAC ducts with elbows, takeoffs, spiral seams, and hanger straps, threaded among joists. Parameters: main-duct diameter, run length, branch/takeoff count, elbow angles, hanger spacing, rectangular-vs-round mix, and dust/decay level. Inspired by pinterest_657103401931589004.jpg.
