# Arch gen: exposed steel post-and-beam interior frame (bpy/bmesh)

## Description

Produces an exposed structural interior bay grid of steel (or encased) columns carrying deep girders and secondary joists, with riveted/bolted connection plates and slung conduit and pipe runs threading between members. Parameters: column grid spacing X/Y, storey height, column and beam section sizes, joist count per bay, and pipe-run density. Inspired by pinterest_657103401931588733.jpg (abandoned factory).
