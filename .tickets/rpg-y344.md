# Arch gen: projecting neon blade signboard (bpy/bmesh)

## Description

Generates a vertical projecting blade sign cantilevered off a facade: a boxed signboard face bordered by a channel, a bracket/outrigger arm bolted to the wall, and a neon-tube glyph layout (or panel) on the face with an emissive material slot. Parameters: board height/width, projection arm length, mounting height, double vs single sided, border depth, and tube-path density.
