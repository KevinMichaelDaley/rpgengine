# Arch gen: mid-century dingbat apartment with tuck-under carport (bpy/bmesh)


## Description

Generates a mid-century Los Angeles 'dingbat' apartment box: a two-story rectangular stucco volume cantilevered over an open ground-floor tuck-under carport with slender steel posts, a flat or low-slope roof, a stucco ribbon fascia carrying a script name plaque, and metal-frame sliding/awning windows in a regular band. Parameters: bay count and carport stall width, floor heights, cantilever overhang depth, post spacing/diameter, window module and sill height, fascia band height, stucco color and weathering. Inspired by pinterest_657103401931588555.jpg (pink 'Florentine' dingbat).
