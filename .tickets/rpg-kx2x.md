# Arch gen: suspended pedestrian skybridge (bpy/bmesh)

Generates an enclosed/open pedestrian skybridge spanning between two buildings: a horizontal deck box with side railings or glazed walls, end abutments plugging into each facade, and optional underslung trusses. Parameters: span length, deck width/height, enclosure type (open rail vs glazed), truss toggle, abutment depth, and support-column drop at midspan. Inspired by pinterest_657103401931588788.png.
