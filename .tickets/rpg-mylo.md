# Arch gen: exterior straight-run apartment access stair (bpy/bmesh)

## Description

Generates an exterior straight-run (single-flight) access stair serving upper-floor units: stringers, open or closed risers, a top landing tying into a walkway, and a pipe or thin-baluster guardrail on both sides. Parameters: rise count, tread depth/riser height, stair width, landing size, railing type (pipe vs wrought-iron), and stringer thickness.
