# Arch gen: recessed streamline arched entry porch (bpy/bmesh)

## Description

Carves a recessed entry porch into a stucco wall with a Streamline-Moderne rounded corner opening (quarter-round vertical jamb, flat lintel), a shallow inset reveal, and a stepped stoop. Parameters: opening width/height, corner fillet radius, recess depth, wall thickness, stoop step count, and an optional wing-wall return.
