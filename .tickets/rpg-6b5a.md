# Arch gen: exposed steel gable roof truss run (bpy/bmesh)

## Description

Builds a repeating run of exposed steel gable roof trusses over a corridor or hall: triangular top/bottom chords with web members and gusset plates, tied by ridge and eave purlins, carrying a light metal roof deck. Parameters: span, pitch, truss count/spacing, web pattern (Pratt/Warren/king-post), chord depth, purlin frequency, and painted-accent color slot. Inspired by pinterest_657103401931588898.png (mall corridor trusses).
