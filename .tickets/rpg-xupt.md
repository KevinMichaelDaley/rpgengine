# Arch gen: stepped ziggurat apartment block (bpy/bmesh)

Generates a stepped ziggurat apartment block: a slab building whose upper storeys set back in terraced steps producing a crenellated stepped silhouette, each terrace with a parapet and ribbon of windows plus wall-mounted AC units and cantilevered balcony pods. Key params: storey count, setback depth per level, step count, window ribbon spacing, balcony pod frequency, AC-unit density, wall color. Inspired by pinterest_657103401931588886.jpg (yellow stepped apartment block).
