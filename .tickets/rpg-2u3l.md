# Arch gen: industrial expanded-metal switchback stair (bpy/bmesh)

## Description

Generates a utilitarian steel egress stair: channel stringers, expanded-metal or checker-plate treads and landings, and welded pipe handrails with mid-rails, optionally boxed inside a corrugated-metal shaft. Parameters: flight count, rise/run, stair width, landing depth, tread type (expanded vs bar-grating vs plate), handrail pipe diameter, and enclosure toggle.
