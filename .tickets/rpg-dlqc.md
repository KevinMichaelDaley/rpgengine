# Arch gen: interior switchback stair with mezzanine landings (bpy/bmesh)

## Description

Produces a multi-level interior switchback (dog-leg) stair connecting stacked mezzanine floors: parallel flights with intermediate half-landings, edged by steel or concrete guardrails and newels, wrapping an open stairwell void. Parameters: number of storeys, flights per storey, rise/run, stair width, landing size, guardrail style, and open-well width. Inspired by pinterest_657103401931588973.jpg (Casino Lisboa) and the graffiti stairwells in pinterest_657103401931588991.heic.
