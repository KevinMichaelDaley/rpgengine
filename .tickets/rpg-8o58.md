# Arch gen: vertical projecting blade and rooftop letter signs (bpy/bmesh)


## Description

Generates tall vertical projecting blade signs and rooftop letter signs: a cantilevered vertical marquee panel with stacked lettering projecting from a facade, plus roof-mounted open channel-letter frameworks on braced steel scaffolds, and an illuminated clock/time sign variant. Parameters: blade height/width and projection, letter row count, rooftop frame truss depth, standoff bracketry, neon-tube vs backlit face, clock module toggle. Inspired by pinterest_657103401931571906.jpg (MAYFAIR / FRANKLIN LIFE rooftop and blade signs).
