# Arch gen: setback art-deco skyscraper crown (bpy/bmesh)


## Description

Generates a setback art-deco skyscraper crown: a tower top that steps back in tiers with vertical pier reveals, a symmetrical ziggurat massing, ornamental buttress fins, and a beacon/finial cap. Parameters: tier count and setback ratio per tier, pier/reveal spacing, base tower footprint, fin depth, crown height fraction, cornice detailing, stone color. Inspired by the Franklin Life deco tower in pinterest_657103401931571906.jpg.
