# Arch gen: riveted plate-girder industrial frame with arched knee braces (bpy/bmesh)


## Description

Generates a heavy riveted-steel industrial frame: built-up plate-girder columns and beams covered in rivet rows, joined by curved knee-brace arches spanning between columns to form an arcade of braced structural bays, as in an early-1900s powerhouse. Parameters: bay count and span, column plate width and rivet pitch, knee-brace arch radius, beam depth, batten/lacing pattern, floor height, rust and paint-loss weathering. Inspired by pinterest_657103401931588741.jpg (abandoned riveted powerhouse).
