# Arch gen: big-box rooftop skylight monitor (bpy/bmesh)

## Description

Generates a large rooftop skylight/roof-monitor opening over a big-box interior: a raised curb framing a rectangular glazed grid supported by a steel space-frame or purlin lattice, with cylindrical pendant fixtures hung below the opening. Parameters: opening length/width, curb height, glazing grid divisions, structural member depth, pendant count/spacing, and a broken-glazing decay ratio. Inspired by pinterest_657103401931588898.png (abandoned mall).
