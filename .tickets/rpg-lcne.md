# Arch gen: utility pole cluster with crossarms and wire tangle (bpy/bmesh)


## Description

Generates a utility pole cluster: timber or concrete poles carrying stacked crossarms with insulators, pole-top transformers, guy-wire anchors, and a tangled web of sagging catenary wires spanning between multiple poles. Parameters: pole count and height, crossarm rows and length, insulator count, transformer count/size, wire sag (catenary droop), guy-wire angle, wire-tangle density, weathering. Inspired by pinterest_657103401931588795.png and pinterest_657103401931588869.jpg (overhead wire poles).
