# Arch gen: gooseneck cobra street lamp (bpy/bmesh)


## Description

Generates a gooseneck / cobra-head street lamp: a vertical pole (or wall bracket) with a curved gooseneck arm terminating in a downturned reflector shade or cobra luminaire head, with an emissive lamp face for glow. Parameters: pole height, arm curve radius and reach, head type (shaded gooseneck vs cobra), shade diameter, mount type (post vs wall-bracket), lamp emissive intensity, patina. Inspired by the streetlamps in pinterest_657103401931571906.jpg and the gooseneck alley lamp in pinterest_657103401931588795.png.
