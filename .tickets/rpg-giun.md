# Arch gen: wrought-iron juliet balcony railing (bpy/bmesh)


## Description

Generates a wrought-iron juliet / shallow balcony railing: an ornamental metal balustrade with scrollwork or plain vertical bars, a top and bottom rail, corner posts, and optional planter hooks and a thin projecting floor lip. Parameters: railing length/height, baluster spacing and profile (scroll vs straight), rail cross-section, projection depth, corner-post style, rust level, plant/hook decoration toggle. Inspired by the plant-laden iron balcony in pinterest_657103401931588841.jpg.
