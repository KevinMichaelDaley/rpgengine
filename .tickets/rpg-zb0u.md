# Arch gen: suspended ceiling grid with fluorescent troffers (bpy/bmesh)

## Description

Builds a suspended lay-in ceiling: an inverted-T grid of main runners and cross-tees holding acoustic tiles, punctuated by recessed rectangular fluorescent troffer fixtures with prismatic diffuser lenses, all hung below a structural deck. Parameters: tile module size, ceiling extent, troffer spacing/frequency, plenum drop height, and a missing/sagging-tile decay ratio. Inspired by pinterest_657103401931588815.jpg (brutalist concourse).
