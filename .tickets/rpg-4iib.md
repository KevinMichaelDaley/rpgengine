# Arch gen: Zigzag steel fire escape with balconies and drop ladder (bpy/bmesh)


Generates a New-York-style zigzag steel fire escape bolted to a facade: stacked landing platforms with railings, angled stair flights connecting alternating sides between floors, and a counter-weighted drop ladder at the bottom. Key params: floor count and storey height, platform width/depth, stair run angle and tread count, baluster spacing, ladder length, standoff bracket depth from wall.
