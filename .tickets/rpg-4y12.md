# Arch gen: circular concrete oculus skylight vault (bpy/bmesh)

Generates a heavy concrete ceiling slab pierced by a row of large circular oculi, each with a deep coffered reveal and a grated cover, suitable for underpass/parking vault ceilings. Parameters: oculus count and spacing, oculus radius, reveal depth, slab thickness, grate pattern density, and support-beam rhythm. Inspired by pinterest_657103401931588576.jpg.
