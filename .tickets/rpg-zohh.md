# Arch gen: rooftop lattice billboard sign frame (bpy/bmesh)

## Description

Generates an open steel-lattice rooftop sign/billboard skeleton: raked support trusses, angled kicker braces bolted to a roof curb, and a rectangular lattice signboard face with a bordering channel where neon letter tubes or panels mount. Parameters: board width/height, number of vertical/horizontal lattice bays, lean-back angle, leg count, and standoff height above roof. Inspired by pinterest_657103401931571985.jpg (Rosslyn sign) and pinterest_657103401931571905.png (Tropicana pole sign).
