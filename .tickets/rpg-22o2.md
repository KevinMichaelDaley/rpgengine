# Arch gen: wrought-iron stairwell gate (bpy/bmesh)

Generates a wrought-iron stairwell gate/guard: an ornamental welded-iron gate with scrollwork and a lower mesh/grille infill mounted in a doorway or over a stairwell descent, with hinge stiles, a top rail, and a mounting frame. Key params: gate width/height, scroll pattern density, mesh-infill fraction, bar diameter, top-rail finial style, single vs double leaf. Inspired by pinterest_657103401931589011.png (STAFF ONLY basement stair gate).
