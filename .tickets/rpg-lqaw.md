# Arch gen: Chain-link cage interior partition (bpy/bmesh)


Generates a chain-link fence cage interior partition: a frame of steel posts and top rails infilled with diamond chain-link mesh, forming a fenced enclosure/cage with an optional gate opening. Key params: run length and height, post spacing, mesh diamond size and wire gauge, gate width/position, top-rail and bottom-tension-bar toggle, sag/deform jitter for a used look. Inspired by pinterest_657103401931589082.jpg (caged stage).
