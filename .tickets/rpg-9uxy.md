# Arch gen: facade drainpipe and conduit run (bpy/bmesh)

Generates vertical downpipes, electrical conduit, and cable runs clamped down a masonry wall, with elbow bends, wall standoff brackets, hoppers, and junction boxes. Parameters: pipe count, per-pipe diameter, run height, bracket spacing, bend/offset points, hopper toggle, and clutter/tangle amount. Inspired by pinterest_657103401931588571.jpg.
