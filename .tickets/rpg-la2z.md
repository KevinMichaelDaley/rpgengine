# Arch gen: stacked freeway interchange (bpy/bmesh)

Generates a multi-level stacked freeway interchange: sweeping curved ramp decks at several stacked elevations, each on rows of tapered concrete piers with pier caps, connected by long-span curved girder segments and edge safety barriers. Key params: number of stack levels, ramp curvature/radius set, deck width and lane count, pier spacing/height per level, girder depth, barrier profile. Should support procedurally routed spline ramps that weave over/under each other. Inspired by pinterest_657103401931572040.jpg (LA interchange aerial).
