# Arch gen: Spanish Colonial clay barrel-tile roof (bpy/bmesh)

## Description

Generates a low-pitch clay barrel-tile (mission-tile) roof: alternating convex pan/cap tile courses running down each slope, a boldly overhanging eave with a tile drip course, hip and ridge cap tiles, and an optional small shed/pergola porch roof. Parameters: pitch, tile radius/overlap, course spacing, eave overhang, hip vs gable configuration, and per-tile jitter for a hand-laid weathered look. Inspired by pinterest_657103401931571998.jpg.
