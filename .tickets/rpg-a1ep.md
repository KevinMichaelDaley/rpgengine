# Arch gen: tiled checkerboard storefront bay with glazed shopfront (bpy/bmesh)


## Description

Generates a ground-floor commercial storefront bay: a recessed glazed shopfront with an aluminum-frame door and large display window, a bulkhead/kickplate below, and a surround clad in a checkerboard ceramic-tile pattern framing the opening. Parameters: bay width/height, glazing subdivision and mullion profile, door position and recess depth, bulkhead height, tile module size and two-color scheme, corner pier width. Inspired by pinterest_657103401931589114.jpg (Triple-Triple checkerboard pizza shop).
