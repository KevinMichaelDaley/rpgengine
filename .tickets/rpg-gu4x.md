# Arch gen: exterior concrete parapet stair (bpy/bmesh)

Generates an exterior poured-concrete stair flanked by solid parapet walls: a straight run (optionally with a landing/dogleg) of concrete treads bounded by raked solid balustrade walls with flat coping, sitting against/into a retaining wall with a stairwell void below. Key params: rise/run counts, tread depth, flight width, parapet wall height/thickness, coping profile, landing option, water-stain/moss seed. Inspired by pinterest_657103401931588886.jpg (Spanish market concrete stair).
