# Arch gen: concrete freeway viaduct deck on bent columns (bpy/bmesh)


## Description

Generates a concrete freeway viaduct / overpass: a deck slab carried on deep longitudinal box or T-girders, supported by tapered bent columns and pier caps, with a guard parapet along the edge and soffit-mounted luminaires; a multi-deck stacked variant is supported. Parameters: span length and count, girder depth/type, column bent spacing and taper, deck width, parapet height, soffit light spacing, number of stacked decks, concrete staining. Inspired by pinterest_657103401931588555.jpg (tuck-under, structural read) and pinterest_657103401931589105.jpg (underpass columns).
