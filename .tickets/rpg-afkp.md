# Arch gen: segmented roll-up garage door bank (bpy/bmesh)

Generates a bank of adjacent roll-up/sectional garage doors set into a wall, each with horizontal segment ribs, a curved head, side guide rails, and stenciled unit numbers. Parameters: door count, per-door width/height, segment (rib) count, jamb/pier width between doors, open/closed state, and number-decal toggle. Inspired by pinterest_657103401931588576.jpg and pinterest_657103401931589092.png.
