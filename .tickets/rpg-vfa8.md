# Arch gen: Electrical meter-box cluster with conduit tangle (bpy/bmesh)


Generates a cluttered exterior electrical service wall: a cluster of rectangular meter/junction boxes at varied depths plus a tangle of conduits and drooping cable bundles running along the wall and ceiling. Key params: box count and size range, cluster region, per-box depth/tilt jitter, conduit run count and sag, cable-bundle thickness, catenary droop, rust/patina variation. Inspired by pinterest_657103401931588838.jpg (meter-box wall) and the wire tangles in pinterest_657103401931588859.jpg.
