# Arch gen: tall multi-pane steel-sash factory window wall (bpy/bmesh)


## Description

Generates a tall multi-pane steel-sash factory window wall: a full-height industrial glazing grid of slender steel mullions and muntins dividing many small lights, with operable pivot/awning vent sections and a masonry reveal surround. Parameters: overall wall width/height, pane grid (rows x columns), mullion vs muntin profile widths, operable-vent rows, broken/missing pane ratio, transom/spandrel bands. Inspired by pinterest_657103401931588741.jpg and pinterest_657103401931589165.webp (industrial hall glazing).
