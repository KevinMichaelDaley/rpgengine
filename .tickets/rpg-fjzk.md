# Arch gen: rooftop TV antenna aerial cluster (bpy/bmesh)


Generates a period rooftop TV aerial cluster: a vertical mast with stacked horizontal Yagi crossbar elements of decreasing length, guy-wire anchors, and a roof-mount base plate. Parameters: mast height, element count and spacing, element length falloff, lean/tilt jitter, and optional satellite-dish attachment. A weathering/bent-element flag adds decay. Inspired by pinterest_657103401931571898.jpg and pinterest_657103401931572033.jpg.
