# Arch gen: wrought-iron window security grille (bpy/bmesh)

## Description

Generates a decorative wrought-iron security grille that mounts over a window or door opening: an outer frame, a lattice of vertical/horizontal bars, and optional scroll/quatrefoil ornament panels. Parameters: opening size, bar spacing and diameter, ornament pattern (scroll, diamond, plain), projection off the wall, and corner mounting-tab toggle. Inspired by pinterest_657103401931571889.jpg and pinterest_657103401931571994.jpg.
