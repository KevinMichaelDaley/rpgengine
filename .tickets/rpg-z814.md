# Arch gen: domed cupola clocktower crown (bpy/bmesh)

Generates a baroque tower crown: a stepped drum carrying a hemispherical/ogee dome with ribs, a lantern finial, and optional clock faces set into a cornice band below. Parameters: drum radius/height, dome profile (hemisphere vs ogee), rib count, lantern height, clock-face count, and finial style. Inspired by pinterest_657103401931588863.jpg.
