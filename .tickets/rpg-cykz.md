# Arch gen: suspended drop-ceiling grid (bpy/bmesh)

Generates a suspended acoustic drop-ceiling grid: a modular T-bar lay-in grid of ceiling tiles hung on wires from a structural deck, interspersed with recessed fluorescent troffers, diffusers, and vents -- with an option to knock out / sag / stain a random fraction of tiles for decay. Key params: room extent, tile module size (e.g. 600/1200mm), troffer density, vent count, hanger drop height, missing/damaged-tile fraction, water-stain seed. Inspired by pinterest_657103401931588716.jpg, pinterest_657103401931588893.jpg and pinterest_657103401931589213.jpg.
