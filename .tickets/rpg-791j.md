# Arch gen: accreted stacked tenement facade with protruding signs (bpy/bmesh)


## Description

Generates an accreted, over-stacked Kowloon-style tenement facade: a base masonry shell overlaid with ad-hoc protruding elements -- cantilevered signboards, cage balconies, boxed-out window enclosures, AC condenser units, drip pipes, and tangled wiring -- built up in irregular strata. Parameters: floor count, protrusion density and depth range, signboard count/size, cage-balcony ratio, AC-unit density, wiring clutter amount, grime/moss weathering gradient. Inspired by pinterest_657103401931588841.jpg and pinterest_657103401931588852.png (dense stacked tenement facades).
