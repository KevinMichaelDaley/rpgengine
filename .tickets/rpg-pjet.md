# Arch gen: Mediterranean-revival stepped apartment block (bpy/bmesh)


Generates a mid-rise Mediterranean-revival apartment block with stepped/tiered massing, arched ground-floor openings, repeated Juliet balconies, and a projecting tile-capped cornice/eave. Parameters: floor count, bay rhythm, setback steps, balcony spacing, arch radius, cornice overhang, and stucco color band. Inspired by pinterest_657103401931571992.jpg.
